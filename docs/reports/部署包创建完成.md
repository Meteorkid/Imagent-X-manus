### 3. 上传部署包
```bash
# 在Mac终端中执行
scp imagentx-deploy.tar.gz root@[您的服务器IP]:/opt/
```

### 4. 在服务器上部署
```bash
# 在服务器上执行
mkdir -p /opt/imagentx
cd /opt/imagentx
tar -xzf /opt/imagentx-deploy.tar.gz
chmod +x *.sh
./quick-deploy.sh --setup
./quick-deploy.sh --deploy
```

### 5. 验证部署
```bash
# 在服务器上执行
./quick-deploy.sh --status

# 在Mac上测试
./check-dns.sh --check
curl -I https://imagentx.top
```
